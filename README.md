# What it is?

### This is a rag based vector search project built from scratch with minimum usage of ai tools. In this I want to explore core techniques such as inverted indexing, TM-IDF, BM25 search, keyword and semantic search, chunking, hybrid search, LLM integration, reranking, evaluation, augmented generation, agentic workflows and multimodal retrieval.

# Useful Commands
### To activate venv: `source .venv/bin/activate`
### To run search command in the cli: `uv run cli/keyword_search_cli.py search "your search query"`
### To run build command in the cli: `uv run cli/keyword_search_cli.py build`
